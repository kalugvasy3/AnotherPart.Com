# AnotherPart.Com

Free in-browser tools. No sign-in, no accounts — your data stays on
your device.

- **/translate** — Voice Translator: speak, read the translation live;
  assemble your own engine pair (recognizer × translator).
- More tools coming: video calls with subtitles, the night sky, an
  audio transcriber.

## Development

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # dist/
```

Deploys to GitHub Pages automatically on push (see
`.github/workflows/deploy.yml`).
